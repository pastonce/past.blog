## D3.js简介

`D3`的全称是`Data-Driven Documents`，顾名思义是一个被**数据驱动**的文档。它本质上是一个`JavaScript`的函数库，可以用来做数据可视化。

[D3.js官方网站](https://d3js.org)

[安利一些不错的D3.js数据可视化资源](https://zhuanlan.zhihu.com/p/385236982)

使用体验上像是在手写`html`页面上的各个元素，页面上显示的每个点、线、面都可以借助该函数库来控制显示与交互逻辑，因此对图像设计的自由度较高，且支持交互和一定程度上的动态变化。

## 安装与使用

* 本地使用：需要在官方网站下载`D3`的压缩包，解压之后将相关的`js`文件包含在相应网页中即可
* 网络加载：直接在相应网页中包含`D3`的网络链接即可，如：

```html
<script src="https://d3js.org/d3.v7.min.js" charset="utf-8"></script>
```

## 简单的可视化流程

用 `D3.js` 进行可视化，可以用矢量图的 `SVG`，也可以用标量图、像素的`canvas`，**可视化画图过程就是将数据映射成视觉元素，再以特定方式布局到画布上**

* 设置画布的宽度、高度、背景颜色等属性
* 绘制图像标题（位置、内容、大小等）
* 设置坐标尺度，即将数据范围映射到网页上指定的长度，绘制坐标轴、刻度、标签、网格（等值线）等
* 绘制数据元素，可以是点、线、区域、箱线等等；绘制时可根据需要设置不同类别数据的显示样式及交互等
* 绘制图例，图例中包含的代表元素尽量与相应数据一致，也可以在图例上设置交互
* 如要绘制多个图片，则需要再画布内合理布局相对位置，且注意根据独立性打组`append("g")`
