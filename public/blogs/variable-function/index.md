## C++变量

> * 编程的大部分内容实际上是在使用数据，而变量可以存储数据并为其进行命名
> * 变量被创建时将会被存储在内存中的堆和栈上
> * **C++不同变量类型之间唯一的区别是占用空间大小**

### 基本数据类型

下表展示了主要数据类型的相关信息，其中**类型大小受编译器影响**

|  数据类型   | 一般类型大小 / bytes |                             描述                             |
| :---------: | :------------------: | :----------------------------------------------------------: |
|   `char`    |          1           |                 用于存储单个字符（或小整数）                 |
|   `short`   |          2           |                              -                               |
|    `int`    |          4           | 在一定范围内存储有符号整数（$2^{-31}\rightarrow 2^{31}-1$）  |
|   `long`    |          4           |                              -                               |
| `long long` |          8           |                              -                               |
|   `float`   |          4           |   单精度浮点数，定义时要在数字后加`f`后缀，否则默认双精度    |
|  `double`   |          8           |                         双精度浮点数                         |
|   `bool`    |          1           | 只有`false(0)`和`true(1)`两个值，实际上仅需一个位，由于字节寻址占用一个字节 |

其中，存储整数的数据类型均可以用`unsigned`修饰，变为无符号整数。

---

可用**操作符**`sizeof`查看数据类型占用字节大小

```c++
#include <iostream>

int main() {
    std::cout << "char: " << sizeof(char) << std::endl;				\\ 1
    std::cout << "short: " << sizeof(short) << std::endl;				\\ 2
    std::cout << "int: " << sizeof(int) << std::endl;						\\ 4
    std::cout << "long: " << sizeof(long) << std::endl;				\\ 4
    std::cout << "long long: " << sizeof(long long) << std::endl;\\ 8

    std::cout << "float: " << sizeof(float) << std::endl;				\\ 4
    std::cout << "double: " << sizeof(double) << std::endl;		\\ 8

    std::cout << "bool: " << sizeof(bool) << std::endl;				\\ 1
    return 0;
}
```

### 浮点数表示

浮点数通常采用IEEE浮点表示，所有位数被切分为符号`s`、尾数`M`与阶码`E`三部分，最终的数值一般为 $(-1)^s \times M\times 2^E$.

* 符号`s`：位于字节表示的最高位，决定浮点数的正负
* 尾数`M`：是一个二进制小数，取值范围为`[0,2)`
* 阶码`E`：对浮点数加权，是2的幂指数

![](/blogs/variable-function/817a098474c0f07a.png)

以常用的单精度浮点数`float`为例，不同的位值可分为以下几种情况：

* 非规格化值：`(exp = 0)` $E = 1 - Bias = -126, M = frac$，表示$\pm 0$以及非常接近$0$的数
* 无穷大：`(exp = 255 & frac = 0)` 表示$\pm \infty$
* `NaN`：`(exp = 255 & frac != 0)` 表示无效值或者未初始化数据
* 规格化值：`(exp != 0 or 255)` $E=exp-Bias,M=1+frac$

其中，$Bias=2^{k - 1} - 1$，对于单精度（$k=8$），该值为 127；`frac`代表的数值取值范围为$[0,1)$.

### 自定义类型

另外，C++ 支持自定义类型，但它们都需要基于上述以及其他一些基本的类型创建。以下简单介绍几种常用的自定义类型，详细特性会在之后的文章中分别深入学习：

* 结构体`struct`：用于将不同类型的数据组合成一个整体，适用于需要同时处理多个相关数据的场景
* 联合体`union`：允许不同成员共享同一块内存空间，适用于需要节省内存或处理不同数据类型的场景
* 枚举`enum/enum class`：用于定义一组命名的整型常量，增强代码可读性，分为传统枚举和强类型枚举两种形式
* 类`class`：是C++的核心特性，支持封装、继承和多态，是面向对象编程的基础
* 类型别名`typedef/using`：用于简化复杂类型声明，提高代码可读性
